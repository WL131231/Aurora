설치 가이드

Windows 설치 방법
1 Aurora.exe 다운로드 후 원하는 폴더에 저장 (설치 프로그램 불필요 — 단일 .exe)
2 처음 실행 시 "Windows의 PC 보호" 경고 → 추가 정보 → 실행 클릭
3 Aurora GUI가 열리면 거래소 API 키를 설정 탭에 입력 후 시작
자동 업데이트: Aurora-launcher.exe를 대신 실행하면 새 버전 자동 다운로드 후 본체를 기동합니다.
macOS 설치 방법
1 Aurora-launcher-macOS.dmg 다운로드 후 더블클릭 → 마운트
2 Aurora-launcher.app 을 Applications 폴더로 드래그 (AppTranslocation 회피 필수)
3 Applications 에서 Aurora-launcher 실행 → "확인되지 않은 개발자" 경고 시 시스템 설정 → 개인 정보 보호 및 보안 → 허용
4 launcher 가 본체 자동 다운로드 + 시작 → 거래소 API 키를 설정 탭에 입력 후 시작
Android 설치 방법
1 알 수 없는 출처 허용 — 설정 → 보안 → "알 수 없는 출처" 또는 브라우저/파일관리자에서 APK 설치 허용
2 위 버튼으로 Aurora-android.apk 다운로드
3 다운로드된 APK 파일 탭 → 설치 버튼 클릭
4 설치 완료 후 Aurora 앱 실행 → 거래소 API 키 입력 후 시작
자동 업데이트: 앱 내에서 새 버전 감지 시 "재시작하기" 버튼으로 바로 업데이트 가능합니다.
Android 8 이상에서는 앱별로 "알 수 없는 출처" 허용이 필요합니다. 브라우저(Chrome 등) 설정에서 해당 브라우저의 설치 권한을 허용하세요.
