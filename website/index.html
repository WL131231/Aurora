Bollinger Bands
01 SELECTABLE · 1H

Bollinger Bands

범위 거래 회귀 매매
  • 4-tier 진입 룰 (Squeeze · Reversal · 찢어짐 · Proximity)
  • 밴드 좁아진 구간(squeeze) 자동 보류
  • 찢어짐 후 회귀 시 강도 1.5 강화 신호
  • 가장자리 안쪽 0.5% Proximity zone 터치 진입
  • 양방향 진입+청산: 상단 → 숏 / 하단 → 롱

RSI Divergence
02 FIXED · 1H

RSI Divergence

모멘텀 반전 신호
  • 4 종류 검출: Regular / Hidden × Bullish / Bearish
  • TradingView Pine Script 표준 (피벗 기반)
  • RSI 수치 자체는 입력값 — 다이버전스만 진입 신호

  • Hidden Divergence → 추세 지속 추가 진입

EMA
03 FIXED · 8 TF

Exponential Moving Average

멀티 TF 지지/저항 터치
  • EMA 200 / EMA 480 — 표준 추세 라인
  • 8 타임프레임 동시 분석 (15m → 1W)
  • 가격이 EMA ±0.3% 이내 터치 시 신호
  • 위에서 지지 → 롱 / 아래서 저항 → 숏
  • HTF 자동 가중치: 1W (25점) → 15m (1점)

Harmonic
04 SELECTABLE · 15m / 1H

Harmonic Patterns

피보나치 정밀 패턴
  • Bat / Butterfly / Gartley 패턴 검출
  • ZigZag 피벗 + 피보나치 비율 검증
  • TP/SL 가격 = 패턴 자체 비율로 자동 산출
  • 다른 지표 대비 진입가·익절가 정밀
  • 구현 예정 — 백테스트로 패턴 정확도 검증

Ichimoku
05 SELECTABLE · 1H / 2H / 4H

Ichimoku Kumo

일목균형표 다중 시간 분석
  • Span A · Span B · Tenkan · Kijun 인지
  • 구름대 두께 = 지지/저항 강도 (두꺼움 → 강함)
  • Span 터치 시 진입, 이탈 시 청산
  • HTF 가중치 자동 (1H=2 / 2H=3 / 4H=5)
