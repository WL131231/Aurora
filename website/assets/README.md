# Aurora Website Assets

## 배경 사진 추가하기

랜딩 페이지에 실제 오로라 사진 배경을 쓰고 싶으면:

1. 원하는 오로라 사진(JPG)을 준비
2. 파일명 `aurora-bg.jpg` 로 이 폴더(`website/assets/`) 안에 저장
3. 페이지 새로고침 → 자동으로 배경에 적용됨

권장 사양:
- 해상도: 1920×1080 이상
- 가로형 (16:9)
- 어두운 배경 + 밝은 오로라 톤 (텍스트 가독성)

파일 없으면 CSS 오로라 띠가 fallback 으로 표시됨 (페이지는 정상 동작).
