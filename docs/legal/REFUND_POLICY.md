# Aurora 환불 정책

**최종 개정일: 2026-05-10**

> ⚠️ **본 정책 측 template 박음.** 실제 적용 전 법무법인 검토 권장.

본 정책은 「전자상거래 등에서의 소비자보호에 관한 법률」 및 「콘텐츠산업진흥법」을 준수합니다.

---

## 제1조 (청약철회)

이용자는 결제 완료 후 다음 조건 하에 청약철회(환불)를 요청할 수 있습니다:

### 가능한 경우 (전액 환불)

1. **결제 후 7일 이내** + **서비스 미사용** (실행 / 매매 미발생)
2. 회사의 귀책사유로 서비스를 정상 이용할 수 없는 경우
3. 회사가 약관을 위반하여 이용자가 손해를 입은 경우

### 제한되는 경우 (환불 거부)

전자상거래법 §17 ② 및 콘텐츠산업진흥법 §27 ②에 따라 다음 경우 환불이 제한됩니다:

1. 이용자의 사용 또는 일부 소비로 인해 가치가 현저히 감소한 경우
2. **이미 본 서비스를 실행 / 매매가 발생한 경우** (디지털 콘텐츠 사용 시작)
3. 시간이 지남에 따라 재판매가 곤란할 정도로 가치가 현저히 감소한 경우
4. 디지털 콘텐츠의 제공이 개시된 경우 (단, 가분적 콘텐츠 일부 미제공 시 그 부분 한정)

⚠️ **본 서비스 측 자동매매 도구로서, 다음 시점부터 "사용 시작"으로 간주됩니다:**
- 본체(.exe / .app)를 실행한 경우
- 봇 시작 버튼을 클릭한 경우
- 거래소 API 키를 등록한 경우

## 제2조 (부분 환불)

월간 / 연간 구독의 경우:

1. 결제 후 7일 이내 + 미사용 — 전액 환불
2. 결제 후 7일 초과 + 미사용 — 사용 가능 잔여 기간에 비례한 환불 (관리 수수료 10% 차감)
3. 결제 후 사용 시작 — 환불 불가 (단, 회사 귀책 사유 시 잔여 기간 환불)

## 제3조 (환불 절차)

1. 이용자는 [aurora-support@example.com](mailto:aurora-support@example.com)으로 환불 요청
2. 회사는 환불 요청 접수 후 **3 영업일 이내** 검토 + 답변
3. 환불 승인 시 **7 영업일 이내** PG사를 통해 결제 수단으로 환불
4. 환불 절차 중 발생하는 PG 수수료, 송금 수수료는 회사 부담 (회사 귀책 시) / 이용자 부담 (이용자 단순 변심 시 — 단 7일 이내 미사용 시 회사 부담)

## 제4조 (분쟁 해결)

환불 관련 분쟁은 다음 절차를 따릅니다:

1. 회사 고객센터 문의
2. 한국소비자원 (https://www.kca.go.kr) 또는 전자거래분쟁조정위원회 (https://www.ecmc.or.kr) 조정
3. 관할 법원 소송 (회사 소재지 관할)

---

**관련 법령**:
- 전자상거래 등에서의 소비자보호에 관한 법률
- 콘텐츠산업진흥법
- 약관의 규제에 관한 법률
